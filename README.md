🟩 MineVault

MineVault es una plataforma web moderna y autogestionable para explorar, compartir y gestionar mods, addons, skins y guías de Minecraft en un solo lugar.

Diseñada con un enfoque mobile-first y una estética inspirada en Minecraft, MineVault ofrece una experiencia fluida, rápida y visualmente atractiva, sin depender de frameworks pesados.

---

✨ Características principales

- 🔍 Búsqueda en tiempo real por título y categorías
- 🏆 Top 10 Mods basado en popularidad
- 🌟 Mod del Mes destacado automáticamente
- 🆕 Contenido reciente con interfaz dinámica
- 📂 Sistema de categorías interactivo
- 📄 Páginas detalladas de mods con descripciones completas
- ⏳ Sistema de descarga con temporizador
- 📚 Sección Wiki con soporte para contenido estructurado
- 🔐 Panel de administración para gestionar contenido fácilmente
- 📱 Diseño totalmente responsive (Mobile-First)

---

🎨 Diseño

MineVault utiliza un diseño moderno en modo oscuro, inspirado en la estética de Minecraft:

- Colores principales:
  
  - Verde esmeralda
  - Gris piedra
  - Negro profundo
  - Acentos dorados

- UI tipo app:
  
  - Animaciones suaves
  - Efectos glow
  - Interacciones dinámicas

---

🧱 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Supabase (Base de datos, autenticación y almacenamiento)

---

🗄️ Funcionalidades del Backend (Supabase)

- Gestión de mods, skins y artículos wiki
- Subida de imágenes
- Sistema de autenticación
- Actualización de descargas en tiempo real

---

🔐 Panel de Administración

MineVault incluye un panel privado accesible desde el navegador que permite:

- Crear y editar mods
- Subir imágenes
- Gestionar categorías
- Escribir contenido tipo wiki
- Administrar todo el contenido sin necesidad de código

---

📱 Enfoque Mobile-First

El proyecto está diseñado principalmente para dispositivos móviles:

- Navegación optimizada para pantallas pequeñas
- Interacciones táctiles fluidas
- Rendimiento optimizado

---

🚀 Objetivo del proyecto

Crear una plataforma moderna, ligera y escalable para comunidades de Minecraft, capaz de competir visual y funcionalmente con portales actuales de mods.
